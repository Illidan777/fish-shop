  УСТАНОВИТЬ ВСЕ ДЛЯ АНГУЛЯРА
	скачать nodejs с офф сайта - https://nodejs.org/uk/download/, установить
 	установить angular cli, запусить в консоли  npm install -g @angular/cli


  ЗАПУСТИТЬ ПРОЕКТ 
	сначала нужно поставить пакеты нужные для проекта, в консоли в папке проекта запустить   npm install
	что бы запустить - в консоли, в папке проекта запусить     ng s
	перейти на  http://localhost:4200
	что бы выключить в консоли Ctrl+C


  ЗАМЕНИТЬ ССЫЛКУ НА СЕРВЕР
	сейчас стоит http://138.68.91.218:8080, если нужно заменить на другой то:
	в проекте, по пути src/environments/environment.ts заменить путь в hostUrl: 'http://138.68.91.218:8080'

	ВАЖНО: файл environment.ts используется только в дев версии, перед билдом проекта нужно также заменить hostUrl в файле environment.prod, он в той же папке лежит

   СБИЛДИТЬ
	в консоли в папке с проектом запустить      ng build --prod 
	в папке проекта должна появиться папка dist
	вот эту папку нужно положить на сервер, указать при переходе на нужный юрл что бы запускался файл index.html 
	ссылка на доки https://angular.io/guide/deployment
